# Windows 11 Hardening Basics

## Account Security
- Enforce strong passwords
- Enable Windows Hello
- Disable unnecessary accounts

## System Protection
- Enable BitLocker
- Configure Secure Boot
- Enable Core Isolation (Memory Integrity)

## Network Hardening
- Disable SMBv1
- Enable firewall for all profiles
- Block inbound connections by default

## Application Control
- Enable Smart App Control
- Restrict PowerShell to Constrained Language Mode (optional)
- Configure App & Browser Control

## Logging & Monitoring
- Enable audit logs
- Configure Event Viewer custom views
- Enable Defender cloud protection

## Technician Notes
- Hardening should not break usability
- Document all changes for rollback
