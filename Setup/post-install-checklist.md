# Post‑Installation Checklist

## Purpose
A quick, technician‑friendly checklist to validate system health, security, and readiness after installing Windows 11.

---

## 1. System Updates
- [ ] Run Windows Update  
- [ ] Install optional security updates  
- [ ] Install driver updates (avoid OEM audio drivers if unstable)  
- [ ] Verify build version is current  

---

## 2. Security Configuration
- [ ] Microsoft Defender real‑time protection enabled  
- [ ] Firewall active for all profiles  
- [ ] SmartScreen enabled  
- [ ] BitLocker or Device Encryption enabled  
- [ ] Tamper Protection enabled  

---

## 3. Privacy Settings
- [ ] Disable advertising ID  
- [ ] Review app permissions (camera, microphone, location)  
- [ ] Disable unnecessary background apps  
- [ ] Disable tailored experiences  

---

## 4. Applications
- [ ] Remove OEM bloatware  
- [ ] Install required software (Office, browsers, tools)  
- [ ] Configure default apps  
- [ ] Install updates for third‑party apps  

---

## 5. System Health
- [ ] Check Device Manager for warnings  
- [ ] Confirm audio, network, and display working  
- [ ] Test speakers, microphone, and webcam  
- [ ] Verify storage health (SMART status)  
- [ ] Create a system restore point  

---

## 6. Documentation
- [ ] Record device name  
- [ ] Record Windows version and build  
- [ ] Store BitLocker recovery key securely  
- [ ] Document any deviations from standard setup  

---

## Technician Notes
- This checklist ensures a consistent, secure baseline across all Windows 11 deployments  
- Use this as part of a standard operating procedure (SOP) for new builds  
- For advanced hardening, refer to **hardening-basics.md**
