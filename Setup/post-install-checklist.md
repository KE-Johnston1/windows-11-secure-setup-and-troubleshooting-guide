
# Post-Installation Checklist

## System Updates
- [ ] Run Windows Update
- [ ] Install optional driver updates (except OEM audio drivers)
- [ ] Verify build version

## Security
- [ ] Defender real-time protection enabled
- [ ] Firewall active
- [ ] SmartScreen enabled
- [ ] BitLocker enabled (if applicable)

## Privacy
- [ ] Disable advertising ID
- [ ] Disable location tracking (if required)
- [ ] Review app permissions

## Applications
- [ ] Install required software
- [ ] Remove OEM bloatware
- [ ] Configure default apps

## System Health
- [ ] Check Device Manager for warnings
- [ ] Confirm audio, network, and display working
- [ ] Create a restore point
