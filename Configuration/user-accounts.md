# User Accounts Configuration

## Purpose
A guide for configuring user accounts securely on Windows 11 systems, including account types, permissions, and sign‑in options.

---

## 1. Account Types
### Microsoft Account (Recommended)
- Enables device recovery  
- Syncs settings (optional)  
- Supports Windows Hello  

### Local Account
- Use only when required  
- Avoid using local admin accounts for daily use  

---

## 2. Administrator vs Standard Accounts
- Use **Standard Accounts** for daily activity  
- Reserve **Administrator Accounts** for system changes  
- Disable or rename the built‑in Administrator account (optional)

---

## 3. Windows Hello
Enable:
- PIN  
- Fingerprint (if supported)  
- Facial recognition (if supported)  

Disable:
- Automatic login  
- Passwordless sign‑in (optional)

---

## 4. Account Security
- Enforce strong passwords  
- Enable sign‑in on wake  
- Disable unnecessary accounts  
- Remove guest accounts  

---

## 5. Family & Other Users
- Remove unused accounts  
- Restrict local accounts  
- Use Microsoft Family Safety for child accounts  

---

## Technician Notes
- Standard accounts reduce malware and misconfiguration risk  
- Windows Hello improves security and user experience  
- Document account changes for auditing
